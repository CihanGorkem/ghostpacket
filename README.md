**Ghost-Packet** is a high-performance, real-time cyber threat visualization tool. It simulates global cyber attack traffic and visualizes it on an interactive 3D-projected world map.

The project demonstrates how to handle high-frequency data streams using **Go** (Backend) and render them efficiently in the browser using **Svelte** and **D3.js** (Frontend).
